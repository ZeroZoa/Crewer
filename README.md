## 프로젝트 목표 및 상세 설명입니다!
러닝으로 이어지는 우리 **Crewer**입니다.<br>
Crewer은 러너들을 위한 커뮤니티이자 운동 기록 저장소입니다.<br>
위치 기반으로 함께 달릴 Crew를 모집하고, 소통하고, 기록하고, 랭킹시스템으로 비교하는 커뮤니티 서비스입니다.<br>
